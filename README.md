# WordPress 主题项目

## 项目简介
这是一个 WordPress 主题项目，包含了 blogmate、blogmatev2 和 minimal-news 三个主题。

## 效果网站
效果网站地址：[www.eluup.com](https://www.eluup.com)

## 主题功能
- 极简新闻主题设计
- 支持 Markdown 内容渲染
- 支持 Mermaid.js 图表显示
- 响应式布局
- 分类排序功能

## 项目结构
- `wp-content/themes/` - 主题目录
  - `blogmate/` - 基础主题
  - `blogmatev2/` - 优化版主题
  - `minimal-news/` - 极简新闻主题
